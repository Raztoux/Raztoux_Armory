- Créez une copie de l'addon "@altisrpfr" situé dans votre répertoire "Steam\steamapps\common\Arma 3" et le nommer (par exemple) "@Fantasmex".

- Aller dedans, rentrer dans le dossier "addons", supprimer les fichiers dont le nom commence par task_force

- Lancez votre ArmA 3, menu principal : configurer => addons => sélectionnez Fantasmex uniquement.

- Placer les dossiers "Fantasmex.Altis" et "Fantasmex_Elpida.xcam_prototype" dans votre dossier répertoire "Steam\steamapps\common\Arma 3\MPMissions"

- Lancer arma 3 et activer le mod @Fantasmex pour profiter du mod Fantasmex sur Elpida.

Enjoy !
